# Pre-Launch Fortune City

## Концепция

Предстартовый период (14 дней до запуска) — юзеры регистрируются, копят Fame, крутят бесплатные спины, покупают замороженные автоматы и приводят рефералов. При старте всё "оживает".

**Главный посыл:** Fame (⚡) будет конвертироваться в $FORTUNE токен после листинга. Чем раньше начнёшь фармить — тем больше получишь. Места в лидерборде ограничены.

---

## 5 механик предстарта

### 1. OG Status

Регистрация в предстарте = **OG Badge навсегда**.

| Что даёт | Описание |
|----------|----------|
| OG Badge | Уникальный бейдж в профиле, никогда больше не будет доступен |
| Приоритет при листинге | OG-юзеры первые в очереди конвертации Fame → $FORTUNE |
| Бонус к конвертации | Множитель при обмене Fame на токен (точный % объявим перед листингом) |

**Cost для проекта: $0** — бейдж бесплатно, конвертация из общего пула.

---

### 2. Открытые тиры 1-3 (ключевой FOMO)

В предстарте **Tier 1, 2, 3 разлочены для всех** — не нужно тратить Fame на unlock.

**В предстарте:**
- Tier 1 (Rusty Lever, $10) — открыт
- Tier 2 (Lucky Cherry, $30) — открыт
- Tier 3 (Golden 7s, $75) — открыт
- Tier 4+ — закрыты (как обычно, нужен Fame unlock)

**После старта:**
- Только Tier 1 доступен по умолчанию
- Tier 2 нужно разлочить за ~200⚡ Fame
- Tier 3 нужно разлочить за ~500⚡ Fame
- Путь: купил Tier 1 → отработал 3 дня → накопил Fame → unlock Tier 2 → купил → отработал 4 дня → ...

**Сравнение: предстарт vs после старта:**

```
ПРЕДСТАРТ (зашёл рано):
День 0: регается, Tier 1-3 открыты
День 0: покупает Tier 3 за $75 (замороженный)
День X (старт): машина включается, через 5 дней → $116.25
Итог: сразу на Tier 3, 0 времени на разлок

ПОСЛЕ СТАРТА (опоздал):
День 0: регается, только Tier 1 открыт
День 0: покупает Tier 1 за $10
День 3: Tier 1 отработал, фармит Fame...
День 5-7: накопил ~200⚡, unlock Tier 2
День 7: покупает Tier 2 за $30
День 11: Tier 2 отработал, фармит Fame...
День 14-16: накопил ~500⚡, unlock Tier 3
День 16: покупает Tier 3 за $75
Итог: Tier 3 через ~16 дней вместо 0. Отставание = 2+ недели.
```

**Почему это работает:**
- Нет бесплатных раздач — юзер платит реальные деньги за машину
- Бонус = экономия ВРЕМЕНИ (не денег) → не бьёт по экономике
- Создаёт жёсткий FOMO: "опоздал = отстал на 2 недели"
- Мотивирует купить сразу Tier 2-3 в предстарте (больше депозитов для проекта)

**Важно:** если юзер КУПИЛ машину Tier 2/3 в предстарте, его maxTierUnlocked сохраняется навсегда. Если только зарегался но не купил — после старта теряет доступ и начинает с Tier 1.

**Cost для проекта: $0** — юзер платит полную цену за машину, бонус только в скорости.

---

### 3. Замороженные автоматы

Покупка автоматов работает уже в предстарте, но машины **заморожены** — не генерируют доход до старта.

**Что работает в предстарте:**
- Покупка автомата за USDT (деньги уходят в казну)
- Реферальные бонусы 5%/3%/1% начисляются (но заморожены)
- Fame за покупку начисляется
- Автомат отображается в зале (визуально "выключен")

**Что НЕ работает:**
- Генерация дохода ($FORTUNE не капает)
- Вывод средств
- Сбор монет, Overclock, Collector

**При старте:**
- Все замороженные машины **включаются с полным циклом** (Tier 1 → 3 дня с нуля)
- Замороженные реф-бонусы размораживаются
- Юзер ничего не теряет — просто старт отложен

**Экономика:**

```
Юзер покупает Tier 1 ($10) в предстарте:
→ $10 сразу в казну
→ Реферер L1: +$0.50 (заморожено до старта)
→ Реферер L2: +$0.30 (заморожено до старта)
→ Реферер L3: +$0.10 (заморожено до старта)
→ Обязательство: $14.50 yield начнёт капать только после старта

Итог: деньги пришли РАНЬШЕ, обязательства начинаются ПОЗЖЕ.
```

**Cost для проекта: отрицательный** — получаем ликвидность до старта.

### 4. Daily Login → Fame Streak

Ежедневный вход в предстарте = Fame. Стрик увеличивает награду.

| День стрика | Fame/день |
|-------------|-----------|
| 1 | 10⚡ |
| 2 | 15⚡ |
| 3 | 20⚡ |
| 4 | 25⚡ |
| 5 | 30⚡ |
| 6 | 35⚡ |
| 7+ | 40⚡ (кап) |

**За 14 дней без пропусков:**

```
Дни 1-7:  10+15+20+25+30+35+40 = 175⚡
Дни 8-14: 40×7                 = 280⚡
Итого:                           455⚡
```

**Что это даёт к старту:**
- Unlock Tier 2 (~200⚡) — можно купить сразу
- Почти unlock Tier 3 (~500⚡) — ещё пару дней
- Кто пропустил предстарт = **0⚡**, отставание = FOMO

**Fame → $FORTUNE токен:** вся Fame, заработанная в предстарте, будет конвертироваться в токен при листинге. Ранние фармеры получат больше.

**Cost для проекта: $0** — Fame не торгуется, не выводится.

### 5. Бесплатные спины колеса → Bonus Fortune

Ежедневные бесплатные спины Wheel of Fortune. Выигрыш копится как **Bonus $FORTUNE**.

**Спины в день:**

| Активных рефералов | Спинов/день |
|--------------------|-------------|
| 0 | 1 |
| 5 | 2 |
| 10 | 3 |
| 15+ | 4 (кап) |

**Математика EV (expected value):**

```
EV одного спина = $0.57 (house edge 43%)

Без рефов (1 спин/день × 14 дней):
→ 14 спинов → EV ≈ $8

5 рефов (2 спина/день × 14 дней):
→ 28 спинов → EV ≈ $16

10 рефов (3 спина/день × 14 дней):
→ 42 спина → EV ≈ $24

15+ рефов (4 спина/день × 14 дней):
→ 56 спинов → EV ≈ $32
```

**Ограничения Bonus $FORTUNE:**
- Можно использовать **только на покупку автомата** (как скидка к цене)
- **Нельзя вывести** в USDT
- **Нельзя крутить колесо** за него
- При покупке автомата вычитается из цены

**Пример использования:**

```
За 14 дней накопил $12 Bonus $FORTUNE.

Покупка Tier 1 ($10):
→ $0 USDT + $10 из бонуса = бесплатно
→ Остаток: $2 Bonus

Покупка Tier 2 ($30):
→ $18 USDT + $12 из бонуса
→ Барьер входа снижен на 40%
```

**Cost для проекта:**

```
Обычный юзер (0 рефов): ~$8 скидка на первую покупку
→ Юзер всё равно покупает автомат → попадает в воронку

Активный рекрутер (15 рефов): ~$32 скидка
→ НО привёл 15 платящих юзеров
→ 15 рефов × $10 мин = $150 депозитов
→ $32 скидка / $150 = 21% CAC — отличный показатель
```

→ К старту: может купить Tier 1-3 сразу без разлока, $8 скидка. Доплата за Tier 1 = **$2**. За Tier 3 = **$67**.

| Механика | Результат |
|----------|-----------|
| OG Badge | Да |

| Замороженная машина | Включится при старте |

| Free Spins | ~$8 Bonus → ушли на покупку |

→ К старту: Tier 2 машина сразу включается. Tier 3 может разлочить за накопленный Fame. Отрыв от тех кто зашёл после старта = **~10 дней**.

### Активный рекрутер (15 рефов, купил Tier 3):

| Механика | Результат |
|----------|-----------|
| OG Badge | Да |
| Tier 3 куплен | maxTierUnlocked = 3 навсегда |

→ К старту: Tier 3 машина включается сразу. Реф бонусы размораживаются. Отрыв от новичков = **~16 дней**.

### Кит (сразу Tier 3 за полную цену):

| Механика | Результат |
|----------|-----------|
| OG Badge | Да |
| Tier 3 ($75) куплен | Включится при старте с полным 5-дневным циклом |
| maxTierUnlocked = 3 | Tier 4 нужно unlock за Fame после старта |

→ К старту: доход пошёл с первой секунды. Через 5 дней — $116.25. Ноль потерянного времени.

## Что нужно реализовать (техническое)

### Приоритет 1 (до запуска рекламы):
1. Флаг `isPrelaunch: boolean` в SystemSettings
2. Блокировка генерации дохода (машины заморожены — создаются но yield не капает)
3. Блокировка вывода средств
4. Заморозка реф-бонусов (начислять, не давать тратить/выводить)
5. Открытые тиры 1-3 в предстарте (maxTierUnlocked = 3 при покупке, или bypass Fame check)
6. Предстартовый баланс "Bonus Fortune" (отдельное поле `bonusFortune` на User)
7. Лендинг с таймером + регистрация

### Приоритет 2 (можно параллельно):
8. OG Badge (поле `isOG` на User, проставляется при регистрации в предстарте)
9. Предстартовые бесплатные спины (выигрыш → bonusFortune, не основной баланс)
10. Daily Fame streak в предстарте (уже работает)
11. Покупка автомата с вычетом bonusFortune из цены

### Приоритет 3 (при переходе со старта):
12. Скрипт разморозки: активировать все замороженные машины (startedAt = now, expiresAt = now + cycle)
13. Разморозка реф-бонусов (перевод из frozen в основной баланс)
14. Закрытие тиров 2-3 для тех кто НЕ купил машину этих тиров (maxTierUnlocked остаётся только у купивших)
15. Уведомления: "Предстарт окончен! Ваши машины включены!"
16. Выключение isPrelaunch → обычный режим работы
